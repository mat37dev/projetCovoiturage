Installation du site:
Placer le fichier index.php et le dossier site à la racine de votre machine (var/www/)
Mettez en place le dump de la base de donnée que vous trouverez dans le dossier structureBDD
Créez un utilisateur avec les permissions suffisantes pour modifier et utiliser la base de donné
Modifier le fichier connexion.php (site/php/connexion.php) avec les valeurs correspondantes.
Votre site est prêt
